# 🚀 Orangopus Project Index

## 📋 Overview
Orangopus is a community-driven development platform built with Vue 3, TypeScript, and Supabase. This document provides a comprehensive index of all components, services, and features.

## 🏗️ Architecture

### Frontend Stack
- **Framework**: Vue 3 with Composition API
- **Language**: TypeScript
- **Router**: Vue Router 4
- **Styling**: CSS3 with modern features
- **Build Tool**: Vue CLI

### Backend Stack
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Real-time**: Supabase Realtime
- **Storage**: Supabase Storage

## 🗄️ Database Schema

### Posts Table
```sql
CREATE TABLE posts (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  user_name TEXT NOT NULL,
  user_avatar TEXT NOT NULL,
  content TEXT NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  likes INTEGER DEFAULT 0,
  tags TEXT[] DEFAULT '{}',
  github_repo JSONB
);
```

### Projects Table
```sql
CREATE TABLE projects (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  title TEXT NOT NULL,
  description TEXT NOT NULL,
  image_url TEXT NOT NULL,
  github_url TEXT,
  live_url TEXT,
  technologies TEXT[] DEFAULT '{}',
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  author_name TEXT NOT NULL,
  author_avatar TEXT NOT NULL,
  slug TEXT UNIQUE,
  category TEXT DEFAULT 'other'
);
```
